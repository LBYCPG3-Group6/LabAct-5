Description of the Game

Black Desert Online is an MMORPG game that has multiple character classes to choose from. Each person may create multiple characters and have the freedom to choose their own classes to play as well as customize the appearance of the character which include facial features and body features.

How to Play

The game is an open world PVP type of game where players can level up and fight with each other.
If PVP is not the type of gameplay for you, then no need to worry as BDO also has a PVE system as well as a Life skilling system which offer playrs an alternative route to generate the game's currency.

Character Classes

Black Desert Online has 22 classes to choose from each with different type of play styles. To go over some of the classes, we will break it down to 7 classes:

Warrior

Warrior is a brawler type class that specializes in close-combat PvP. They utilize their protection skills while dealing heavy damage to their opponents.

Witch

Witch is a mage type class that specializes in using spells to do heavy AOE damage whilst being able to provide sustain for the team.

Guardian

Guardian is a bruiser type class that does heavily protected skills whilst providing powerful AOE damage.

Ranger

Ranger is a ranged archer type class that specializes in long range combat.

Sorceress

Sorceress is a spellcaster that has a lot of short-ranged burst of movements whilst having ample amounts of long-ranged damage.

Shai

Shai is a support type class that specializes in healing and applying buffs to teammates whilst also having skills to disrupt enemies.

Hashashin

Hashashin is an assassin type class that specializes is doing strong bursts of damage to opponents whilst being able to quickly escape.

Game Rules

The rules of the game are fairly simple:

  2. Don't promote hate speech of any kind.
  3. Be friendly.

How the game is played is entirely up to the player on which route they wish to take.
